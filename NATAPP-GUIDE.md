# NATAPP 免费隧道配置指南

## 注册步骤

### 1. 访问 NATAPP 官网
打开浏览器访问：https://natapp.cn/

### 2. 注册账户
点击页面上的"注册"按钮：
- 使用邮箱注册
- 设置密码
- 完成验证

### 3. 实名认证
- 根据页面提示完成实名认证（国内服务需要）
- 认证后即可使用免费隧道

### 4. 购买/领取免费隧道

登录后：
1. 进入"我的隧道"页面
2. 点击"购买隧道"
3. 选择"免费隧道"（有流量限制但够用）
4. 选择隧道类型：**Web隧道**
5. 隧道名称：填写 `autosight`
6. 本地地址：`127.0.0.1`
7. 本地端口：`5000`

### 5. 下载客户端

在 NATAPP 下载页面下载 macOS 客户端：
- https://natapp.cn/#download

选择 **macOS ARM64** 版本（因为你是 Apple Silicon Mac）

### 6. 配置认证

下载后，需要配置 authtoken：

1. 登录 NATAPP 网站
2. 进入"我的隧道" → "免费隧道"
3. 复制 **authtoken**（一串字符）

创建配置文件 `~/.natapp/config.ini`：

```ini
[client]
authtoken=你的authtoken
```

### 7. 启动隧道

在终端中运行：

```bash
# 添加执行权限
chmod +x ~/Downloads/natapp

# 运行
~/Downloads/natapp
```

如果下载到了其他位置，请相应调整路径。

### 8. 获取公网地址

成功启动后，你会看到类似这样的输出：

```
 Tunnel Status   : Online
 Account          : free (未支付)
 Forwarding       : http://xxxxx.natappfree.cc -> 127.0.0.1:5000
```

**你的公网访问地址就是：** `http://xxxxx.natappfree.cc`

## 注意事项

1. **免费隧道限制**：
   - 每月有流量限制
   - 隧道可能随时变化（重启后）
   - 适合测试和开发

2. **保持运行**：
   - 关闭终端后隧道会断开
   - 需要保持终端打开或使用 nohup 后台运行

3. **安全性**：
   - 免费隧道没有 HTTPS
   - 敏感操作建议使用付费隧道

## 常见问题

**Q: 显示"认证失败"？**
A: 检查 authtoken 是否正确复制，可能有多余空格

**Q: 隧道显示"离线"？**
A: 检查你的网络连接，或 NATAPP 服务是否维护

**Q: 访问很慢？**
A: 免费隧道使用共享带宽，高峰期会慢，付费可改善
